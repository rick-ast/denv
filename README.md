# denv

### Configuration files

- `rick.terminal` is the configuration file for Terminal App on MacOS
- `rick.zsh-theme` should be under `$HOME/.oh-my-zsh/custom/themes` for [Oh My Zsh](https://ohmyz.sh), then change `ZSH_THEME` to "rick" in `.zshrc`
