# Environment Setup

1. Install [Homebrew](https://brew.sh/). \
   use directive printed to console to config env after installation
2. `brew install git`
3. `brew install node@<LTS>`
4. `brew link node@<LTS>`

to use `pnpm`

```sh
npm i -g pnpm
```

not recommended currently

```sh
corepack enable pnpm
```
